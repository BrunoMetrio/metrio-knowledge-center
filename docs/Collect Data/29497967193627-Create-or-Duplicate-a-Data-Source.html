Create or Duplicate a Data Source

Updated January 09, 2025 21:47

Create a new data source

  1. Click + New data source in the top right corner of the window.
  2. Fill out required info: Data source name, Data source ID, Category.
  3. Click on Create new data source.

Duplicate an existing data source

  1. From an existing data source, click on (More) ⋮  > Duplicate
  2. Fill out required info: Data source name, Data source ID.
  3. Click on Duplicate data source.
